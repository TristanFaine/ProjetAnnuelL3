Ce scrapper s'utilise avec un bot discord, ceux-ci sont configurables a cette adresse : https://discord.com/developers/applications/

Cette methode de scrapping possede donc quelques limitations, seules les données a la portee du bot sont disponibles, il faut d'abord inviter le bot dans un serveur avant de pouvoir recuperer les données de celui-ci.
